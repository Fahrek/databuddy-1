#

elevator.py implements 2 elevator modes - Mode A and mode B

#
#

Execute mode A:
	python elevator.py -f input.txt -m A
	
Execute mode B:
	python elevator.py -f input.txt -m B
	
Input arguments:

	-f : text file with elevator inputs
		e.g. 9:1-5,1-6,1-5
		
	-m : Calculation mode (A or B)
	

Test:
	python elevator_lib.py
